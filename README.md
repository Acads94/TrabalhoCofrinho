# Cofrinho de Moedas - Java (Atividade Uninter)

Sistema desenvolvido em Java com foco na aplicação de conceitos de **Programação Orientada a Objetos (POO)**. O projeto simula um cofrinho virtual que gerencia moedas de diferentes nacionalidades.

#Funcionalidades
* **Adicionar Moedas:** Suporte para Dólar, Euro e Real.
* **Remover Moedas:** Retirada de itens específicos da coleção.
* **Listagem:** Exibição de todas as moedas presentes no cofrinho.
* **Conversão Total:** Cálculo do saldo total convertido para Real (BRL).

#Conceitos de POO Aplicados
O projeto utiliza os pilares da programação orientada a objetos para garantir um código limpo e reutilizável:
* **Abstração:** Classe base abstrata `Moeda`.
* **Herança:** Especializações para cada tipo de moeda.
* **Polimorfismo:** Tratamento genérico das moedas dentro da coleção.
* **Encapsulamento:** Proteção dos dados e lógica interna das classes.

#Tecnologias e Ferramentas
* **Linguagem:** Java
* **Estrutura de Dados:** `ArrayList` para gerenciamento da coleção.
* **Ambiente de Desenvolvimento:** Eclipse IDE (Original) / VS Code (Exportação).
* **Interface:** Console (Interativa).
